Projects

Projects

Below are some of the projects I have worked on, during my bachelor's academics and internship tenure in the software engineering domain.

eBPM Portal

The eBPM Portal is a full stack web application developed to streamline business process management by digitizing and automating task workflows within an organization. Built using FastAPI (Python) for the backend and Angular for the frontend, the portal enables efficient task assignment, tracking, and role-based access management.

Key features include CRUD-based user and task management, group and role assignment, and an intuitive UI for managers and employees to monitor individual, group, and reportee tasks. Data is securely stored in MongoDB, with real-time updates facilitated by FastAPI’s RESTful services. The portal also includes robust logging via a task history module, helping maintain transparency and traceability.

Designed during a Practice School internship at EAIESB Software Solutions, this project emphasizes minimizing manual intervention, improving organizational efficiency, and delivering a modern, user friendly experience for process management.

Grab Online Order [GOO]

Grab Online Order (GOO) is a mobile and web based food ordering platform developed to eliminate unnecessary delivery charges and reduce customer wait times. Unlike traditional food delivery apps, GOO allows users to place orders from nearby restaurants and pick them up directly enabling cost savings and timely access to meals.

Designed during a foundational entrepreneurship course, GOO was validated through customer surveys and feedback from over 39 users, with 86% recognizing the core problem. The solution offers features like real time order notifications, digital payments, and flexible pickup options, making it ideal for students, bachelors, and working professionals who seek affordability and convenience.

By removing the dependency on delivery agents and avoiding extra fees, GOO empowers both customers and restaurants while delivering a scalable and sustainable food ordering experience.

FlyEasy

Fly Easy is a smart, AI powered web application developed to help students find the most suitable universities abroad based on their academic profiles, preferences, and constraints. The platform leverages machine learning algorithms and web scraping techniques to recommend graduate programs tailored to the user’s qualifications, budget, location preferences, and desired course of study.

The system was built as part of an bachelor's academic capstone project and presented at the ICCCN 2022 conference. It collects real time university data, ranks institutions using a weighted scoring model, and presents personalized suggestions to students through a clean and user friendly interface.

By combining intelligent filtering, profile analysis, and dynamic data retrieval, Fly Easy significantly reduces the time and effort students typically spend researching options offering them a guided path to make informed, data backed decisions about their higher education.

Crypto Price Tracker

Crypto Price Tracker is a responsive and modern web application built with Next.js, designed to display real time cryptocurrency prices using the CoinGecko API. The app offers an interactive interface that allows users to search, view, and refresh live market data for the top five cryptocurrencies by market cap.

With a strong focus on performance and user experience, the project leverages React Query for efficient state and data management, Axios for API communication, and Bootstrap for styling and responsive design. It includes features like live price refresh, search functionality, loading indicators, and error handling for a polished, production-ready experience.

Additionally, the project includes full Docusaurus based documentation, making it easy for developers to understand, contribute, and deploy. This project reflects a complete setup from backend API integration to frontend development and documentation showcasing professional full-stack development practices.
